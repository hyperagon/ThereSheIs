# There She Is!!
Files to setup [Ruffle](https://ruffle.rs/) to play **There She Is!!**.

You can [watch the whole thing here](https://www.youtube.com/watch?v=3quYcNI3dOs) but even HD isn't as good as the vector originals since those can scale to **any size**.

While I [downloaded the **SWF** files](https://eye.swfchan.com/search/?q=there+she+is), they belong to their respective owners: [SamBakZa](https://sambakza.net/).

(You can find the creators (SamBakZa) on [Newgrounds](https://sambakza.newgrounds.com/movies), [Youtube](https://www.youtube.com/@amalloc/videos) or [their website](https://shopsambakza.net/).)

Use `python -m http.server` and [enjoy it](https://youtu.be/3quYcNI3dOs?t=1404).
